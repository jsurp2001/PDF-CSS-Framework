Markdown Cheat Sheet

Your comprehensive guide to writing Markdown, including advanced features.

1. Headings

Use # for headings. The number of # determines the heading level (1-6). 

# Heading 1
## Heading 2
### Heading 3

2. Text Formatting

  • **Bold**: Bold
  • *Italic*: Italic
  • ~~Strikethrough~~: Strikethrough
**Bold** *Italic* ~~Strikethrough~~

3. Lists

Unordered List

- Item 1
- Item 2
  - Subitem 2.1

Ordered List

1. Item 1
2. Item 2
   1. Subitem 2.1

4. Links and Images

Links

Use [Text](URL).

[GitHub](https://github.com)

Images

Use ![Alt Text](Image URL).

![Markdown Logo](https://markdown-here.com/img/icon256.png)

5. Code

Inline Code

`Inline code example`

Code Block

```html

Hello, Markdown!

```

6. Blockquotes

> This is a blockquote.
>> It can span multiple lines.

7. Horizontal Rule

Use ---, ***, or ___.

---

8. Tables

| Header 1 | Header 2 |
|----------|----------|
| Row 1    | Data     |
| Row 2    | Data     |

9. Task Lists

- [x] Completed Task
- [ ] Incomplete Task

10. Footnotes

Use [^1] for inline footnotes.

Here is a sentence with a footnote.[^1]

[^1]: This is the footnote text.

11. Emojis

Use emoji shortcodes like :smile:.

I :heart: Markdown! :smile:

12. Collapsible Sections


  Click to expand
  Hidden content here.
